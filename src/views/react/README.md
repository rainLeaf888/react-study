## diff 算法
  * 算法复杂度 O(n) 深度优先
## diff 策略
1. 同级比较, web ui 中Dom节点跨层级移动操作很少，可忽略不计
2. 两个不同类型的元素会生成不同的树

  例如：A --> B, CompA --> CompB
3. 开发者可以通过 key prop 来暗示哪些子元素在不同的渲染下能保持稳定（保证key的稳定性）

## react fiber
* 能够把可中断的任务切片处理。
* 能够调整优先级，重置并复用任务。
* 能够在父元素与子元素之间交错处理，以支持 React 中的布局。
* 能够在 render() 中返回多个元素
* 更好地支持错误边界

## fiber 结构
* child
* sibling
* return
